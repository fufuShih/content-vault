# Content Vault - Server

## Technologies Used

- Node.js
- Express.js
- TypeScript
- MongoDB
- Mongoose
- JWT Authentication
- Bcrypt
- Cors
- Dotenv

## Development Setup

1. Install dependencies:
   `npm install`

2. Create .env file with required environment variables

3. Run development server:
   `npm run dev`

4. Build for production:
   `npm run build`
