# Content Vault - Client

## Technologies Used

- React.js
- TypeScript
- Vite
- TailwindCSS
- React Query
- React Router DOM
- Axios
- DaisyUI

## Development Setup

1. Install dependencies:
   `pnpm install`

2. Run development server:
   `pnpm run dev`

3. Build for production:
   `pnpm run build`
